## Usage

Make sure to include the web component bundle within the `<head>` section of your html.

<Description markdown={vars.cdnUrl} />

```html
<script src="kyndryl-web-components.bundle.js"></script>
<link src="kyndryl-web-components.css" rel="stylesheet" />
```

### Markup

Place the custom web component tags into the `<body>` of your html.

```html
<kyndryl-header></kyndryl-header>
```
